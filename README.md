# 闲鱼虚拟资料网盘发货检查器

一个免费开源的浏览器端小工具，用于闲鱼虚拟资料、模板、教程、Excel、PPT 和提示词资料包卖家在发送网盘链接或自动发货信息前做检查。

Live demo:

https://ronnie2025.github.io/xianyu-netdisk-delivery-checker/

## 解决什么问题

低价虚拟资料卖家常见的售后问题不是不会发链接，而是发出前没有把这些边界检查清楚：

- 网盘链接、提取码、权限和有效期没有测试。
- 文件名、版本号、文件格式和打开方式不清楚。
- 商品详情页没有写明电子资料无实物邮寄。
- 链接失效、漏发、文件损坏时没有补发规则。
- 退款边界写得含糊，容易把使用问题变成交易争议。
- 自动发货后没有订单记录，无法判断低价资料包是否真的赚钱。

这个工具会根据表单和勾选项生成：

- 发货前自查清单
- 给买家的网盘说明
- 售后和退款边界
- 订单记录字段
- 可下载的 Markdown 检查清单

## 本地使用

这个项目是纯静态页面，不需要后端服务。

```bash
open index.html
```

也可以用任意静态服务打开：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000/`。

## 文件

- `index.html`: 主检查器页面。
- `xianyu-netdisk-delivery-checklist.md`: 可直接复用的 Markdown 清单。
- `sitemap.xml`: GitHub Pages 站点地图。
- `robots.txt`: 搜索引擎抓取声明。
